# Frontend 

Our frontend application is written in reactive framework [Vue](https://vuejs.org).

## Table of contents


1. [Prerequisites](#prerequisites)

### Installation and development

1. Installing dependencies

    ```shell script
    npm install
    ```
   
1. Running the dev server

   ```shell script
   npm run serve
   ```
